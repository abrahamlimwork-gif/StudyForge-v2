# **App Name**: StudyForge

## Core Features:

- User Authentication: Secure user login and registration using Firebase Auth with Google and Email Link options.
- Dashboard Page: Personalized landing page for logged-in users to view upcoming sessions and access resources.
- Classroom Page: Dedicated page for live teaching sessions, integrating a video conferencing tool within a div with id 'jitsi-container'.
- Session Management: Manage and store session room names in a 'sessions' Firestore collection.
- Jitsi Video Integration: Facilitate live video conferencing within the Classroom page for interactive teaching.
- Lesson Prompt Generator: AI tool to generate thought-provoking discussion prompts for lessons based on a given topic or scripture reference, to aid instructors in leading engaging sessions.

## Style Guidelines:

- A sophisticated deep indigo (#3B2666) is chosen as the primary color, embodying wisdom and spiritual depth for a teaching platform. This provides a strong contrast against lighter elements, optimizing readability for older users.
- The background color is a very light lavender gray (#F7F5FA), a desaturated variant of the primary hue, providing a clean and serene canvas that's easy on the eyes while subtly harmonizing with the brand color.
- A vibrant and clear sky blue (#6398F6) is selected as the accent color. This color, analogous to the primary, offers high visibility and serves effectively for interactive elements like high-contrast buttons, guiding user attention without being distracting.
- Headline and body font: 'PT Sans', a humanist sans-serif. It combines a modern aesthetic with warmth and excellent readability, which is ideal for a teaching platform and legible at large font sizes, catering to older users.
- Simple, universally recognized icons with a clean, high-contrast silhouette. Emphasize clarity and direct communication over ornate designs to enhance user comprehension.
- Extremely simple, clean layouts with generous spacing and large, easily readable text. Utilize clear hierarchies, direct navigation, and high-contrast buttons to minimize cognitive load and enhance accessibility for older users.
- Subtle, deliberate animations focused on reinforcing user actions or providing clear feedback. Avoid excessive or rapid motion to maintain a calm learning environment and prevent user disorientation.
